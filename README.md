## Setup

### clone git repository

```bash
git clone "https://github.com/cardo-usa/usa/"
```

### launch conatiner

```bash
docker compose -f "./docker/docker-compose.development.yaml" -p "usa" up -d
```

### set env variables

See `.env.example` for more details.
